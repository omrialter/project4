Toys API documentation

to get toys

Explanation METHOD URL
to get the first 10 toys GET https://omrialter.cyclic.app/toys
to get toys per page(in the example we try page 2) GET https://omrialter.cyclic.app/toys/?page=2
to sort the results by property(in the example we try to sort by price) GET https://omrialter.cyclic.app/toys/?sort=price
to reverse the order of the results (in the example we reverse the results of the price sort) GET https://omrialter.cyclic.app/toys/?sort=price&&reverse=yes
to search for results by name of toy or by the information of the toy(in the example we try to search a toy with the word pazel in the name or info ) GET https://omrialter.cyclic.app/toys/search/?s=pazel
to get a list of toys from a specific category(lets say sport category) GET https://omrialter.cyclic.app/toys/category/?s=sport
to get a single toy by his id (in this example you need to change the "toy_id" with the actual id of the toy) GET https://omrialter.cyclic.app/toys/single/toy_id

to post a new toy

in order to post a toy you must log in with a user and you must send a valid token

Explanation METHOD URL
to create a new toy: POST https://omrialter.cyclic.app/toys
the schema to create a new toy:
name: String,
info: String,
category: String,
img_url: String,
price: Number

to edit a toy

in order to edit a toy property you need to send a valid token

Explanation METHOD URL
to edit a toy(in this example you need to change the "toy_id" with the actual id of the toy) PUT https://omrialter.cyclic.app/toys/toy_id

to delete a toy

in order to delete a toy you must send a valid token

Explanation METHOD URL
to delete a toy(in this example you need to change the "toy_id" with the actual id of the toy) DELETE https://omrialter.cyclic.app/toys/toy_id

Sign up

Explanation METHOD URL
to sign up: POST https://omrialter.cyclic.app/users
the schema to create a new user:
name: String,
email: String,
password: String

Log in:

Explanation METHOD URL
to log in POST https://omrialter.cyclic.app/users/login
the schema to log in:
email: String,
password: String

to get your user info:

Explanation METHOD URL
to get your user info (you need to send valid token) GET https://omrialter.cyclic.app/users/userinfo
